# Dependency advisory inventory — 2026-07-12

Lockfile: `package-lock.json`

## Result

`npm audit` initially reported 15 vulnerable dependency nodes (1 critical, 6 high, 8 moderate). A non-forced lockfile refresh removed every advisory except the Anthropic filesystem-memory permission advisory; upgrading the three direct `@mariozechner/pi-*` packages from 0.70 to 0.73 removed that final transitive advisory. The final audit reports zero known vulnerabilities.

The `@mariozechner` packages now emit a deprecation notice recommending the renamed `@earendil-works` packages. That rename is not a security advisory and is deliberately not mixed into this bounded upgrade.
