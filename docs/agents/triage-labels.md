# Triage labels

| Canonical role    | Local status      | Meaning                             |
| ----------------- | ----------------- | ----------------------------------- |
| `needs-triage`    | `needs-triage`    | Requires evaluation                 |
| `needs-info`      | `needs-info`      | Requires Owner input                |
| `ready-for-agent` | `ready-for-agent` | Fully specified and safe to execute |
| `ready-for-human` | `ready-for-human` | Requires hands-on human work        |
| `wontfix`         | `wontfix`         | Intentionally not planned           |

Completed local tracers use `complete`. Active tracers use `in-progress`.
