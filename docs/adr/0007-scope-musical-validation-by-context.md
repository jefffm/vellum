# Scope musical validation by context

Vellum will evaluate counterpoint, voice leading, dissonance treatment, continuo, and related musical behavior through explicit Validation Profiles selected from the passage's Texture, Contrapuntal Techniques, historical scope, and task. It will not treat generic prohibitions such as parallel perfect intervals as universal errors.

## Considered options

- Apply one universal lint rule set to every score
- Make all findings advisory and leave adjudication to the model
- Classify findings as hard constraints, soft preferences, or observations under a named contextual profile

## Consequences

Every Validation Finding must cite its applicable profile and supporting Historical Practice Claims. Profile selection becomes part of Musicological Analysis and remains correctable. Existing blanket lint results are low-level observations until a Validation Profile determines their musical consequence.
