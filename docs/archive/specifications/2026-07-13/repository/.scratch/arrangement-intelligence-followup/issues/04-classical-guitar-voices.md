# Coherent six-string classical-guitar target-voice compiler

Status: ready-for-agent

Type: AFK

## Tracer

Reduce the reviewed Greensleeves SATB source to a deliberate classical-guitar target texture. Preserve the upper Principal Voice and create a coherent phrase-level bass derived from the source bass and harmonic/cadential analysis, with optional inner filling subordinate to both planned voices.

## Acceptance criteria

- The default reviewed candidate is an explicit two-voice target texture, not four sparse source-part remnants.
- A coherent source bass is structurally retained or transformed with declared octave, rhythm, omission, and lineage decisions; a generated bass is used only when Analysis and Plan authorize composition.
- Search tracks both voices' pitch, onset, duration, release, register, cadence goals, and physical positions jointly.
- Hard constraints protect Principal Voice, required structural bass/cadence events, voice order, intentional sustain, and simultaneous playability.
- Contextual evaluation covers bass continuity, unjustified silence, registral and interval coherence, cadence completion, crossings, applicable dissonance treatment, harmony, and left/right-hand feasibility.
- Standard notation engraves intentional upper and lower target voices with correct stems, rests/skips, ties, and written/sounding pitch semantics.
- Audio Preview isolates upper voice, bass, and optional inner filling.
- The Greensleeves regression rejects the current 71-upper/4-bass outcome.
