# Shared phrase intent, target voices, ergonomics, and counterexamples

Status: ready-for-agent

Type: AFK

## Tracer

Carry one reviewed phrase from Analysis and Arrangement Plan through an explicit Intended Technique Plan, proposed Target Voice Plan, Owner Ergonomic Profile, serializable search state, candidate report, and output evaluator. Preserve target-specific extension points rather than defining one generic fretted-instrument heuristic.

## Acceptance criteria

- Intended technique and target texture are passage-scoped Plan decisions with evidence, alternatives, and uncertainty.
- Target voices have durable identities, roles, onset/duration continuity, cadence obligations, allowed transformations, and source lineage.
- Instrument mechanics remain universal facts; Intended Performer and Owner Ergonomic Profiles supply contextual thresholds without rewriting mechanics.
- Physical reach is measured from exact scale length and fret geometry where applicable.
- Search cannot hide an impossible span by clamping finger numbers.
- The three Owner counterexamples are versioned output-level regression fixtures.
- Candidate reports explain why locally convenient choices lose to coherent phrase-level alternatives.

## Honest boundary

The tracer defines executable contracts and minimal cross-target examples. It does not assert that a target-specific idiom is complete before issues 2–4.
