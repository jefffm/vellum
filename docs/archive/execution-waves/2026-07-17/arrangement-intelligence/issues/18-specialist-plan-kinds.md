# Creative and specialist Plan kinds

Status: complete

Type: AFK

User stories: U3, U8

## What to build

Extend planning to creative arrangement, continuo realization, and imitative intabulation with material alternatives and explicit conflicts.

## Acceptance criteria

- [x] Each Plan kind defines required and optional decisions without flattening its musical domain.
- [x] Materially different plans produce materially different target candidate families.
- [x] Target infeasibility creates a Plan Conflict rather than silent relaxation.
- [x] Plan alternatives retain evidence and Owner confirmation independently.

## Completion evidence

- Arrangement Plan is now a closed five-kind contract. A kind-matched specialist-intent union retains creative formal/textural/harmonic/idiomatic design, exact Continuo Realization Profile/Foundation/figure/voice-leading intent, or imitative voice-distribution/overlap/entry/cadence intent without projecting those domains into generic strings alone.
- Store validation resolves every specialist target and decision against the exact Analysis and Target Configuration. Creative Plans require generated material owned by an explicitly confirmed Owner decision; continuo Plans require the target's exact Realization Profile and Continuo Foundation targets; imitative Plans require ordered-entry and cadential-goal targets.
- Explicit `free_paraphrase` now produces a confirmed creative Plan with separately evidence-bearing alternatives. Its ornamented-paraphrase candidate adds playable generated musical material; its idiomatic-revoicing candidate remains materially distinct.
- A corrected immutable creative Plan can restrict realization to a different candidate strategy. Passing its exact Plan ID produces a new candidate family and different Arrangement Score; generic ranking no longer silently overrides the Plan-selected candidate.
- Continuo and imitative generation consume only the candidate strategies declared by their Plan. Fixtures prove complete versus lean/separate-bass/reduction families and low-fret versus voice-continuity families contain materially different events while retaining exact profile, foundation, entries, cadences, evidence, alternatives, policy consequence, and confirmation state.
- An impossible faithful continuo target no longer falls through or silently relaxes. It persists a typed Plan Conflict, returns API code `plan_conflict`, lists all five explicit resolution paths, marks the search failed, and creates no Arrangement Score.
- Selecting a conflict resolution moves it only to `resolution_selected`; it cannot falsely claim the target-local/shared Plan revision, policy change, exception, or block has already been executed. A second silent selection is rejected. T27 owns execution of family/target resolution paths.
- Workspace migration and recovery include durable Plan Conflict identities. Full regression: 987 passed, one expected provider smoke skip.
- Retained machine-readable verification: `.scratch/arrangement-intelligence/evidence/T18/verification.json`.
