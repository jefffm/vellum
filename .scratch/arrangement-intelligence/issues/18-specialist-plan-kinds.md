# Creative and specialist Plan kinds

Status: ready-for-agent

Type: AFK

User stories: U3, U8

## What to build

Extend planning to creative arrangement, continuo realization, and imitative intabulation with material alternatives and explicit conflicts.

## Acceptance criteria

- [ ] Each Plan kind defines required and optional decisions without flattening its musical domain.
- [ ] Materially different plans produce materially different target candidate families.
- [ ] Target infeasibility creates a Plan Conflict rather than silent relaxation.
- [ ] Plan alternatives retain evidence and Owner confirmation independently.
