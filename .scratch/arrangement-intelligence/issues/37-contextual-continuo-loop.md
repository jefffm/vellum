# Contextual continuo through the shared loop

Status: ready-for-agent

Type: AFK

User stories: U3, U6, U8

## What to build

Validate the shared architecture with a non-position-search domain that realizes connected continuo voices under exact historical and bass authority.

## Acceptance criteria

- [ ] The plan distinguishes complete realization, separate bass, and honest reduction.
- [ ] Unfigured and figured bass use contextual harmony, alterations, voice leading, and prepared resolutions.
- [ ] Generated voices remain distinct from source evidence and satisfy complete audit and playback semantics.
- [ ] Cross-domain Evaluation Cases reuse shared contracts without flattening continuo state.
