# Passage-level Musicological Analysis

Status: ready-for-agent

Type: AFK

User stories: U2, U3, U8

## What to build

Produce correctable, evidence-bearing passage Analysis sufficient for planning rather than one global texture or crude register heuristic.

## Acceptance criteria

- [ ] Passage boundaries, Texture, roles, phrases, cadences, and Contrapuntal Techniques are explicit.
- [ ] Principal Voice inference records musical evidence and material alternatives.
- [ ] Imitation searches delayed interval-rhythm entries beyond initial notes.
- [ ] Analysis uncertainty feeds Source Truth and Arrangement Planning deterministically.
